# TabRegistration

本场景用于声明作者可见的自定义分页。

## 适用时机

- 模组需要在物品、配方或制造器相关界面投射一个自定义分页。
- 分页可以由稳定 ID、标题、图标和排序描述。

## 关键参数

- `Id`：分页稳定 ID。
- `OwnerModGuid`：归属模组。
- `Title`：分页标题。
- `IconId`：图标 ID。
- `Order`：排序权重。

## 运行时前提

在启动阶段注册一次。当前已支持的投射面有限；全息信标、蓝图、信号选择器等界面需要更完整内容模型后再支持。

## 常见误用

- 不要把分页内容和复杂 UI 状态塞进 descriptor。
- 不要假设所有 DSP UI 表面都已支持自定义分页。

代码示例见 `TabRegistrationExample.cs`。
